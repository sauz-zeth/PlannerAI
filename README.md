# Smart Planner 🗓️

> Voice & Text Scheduling for Apple & Google Calendar

## 🎯 **What We're Building**

A smart assistant that lets you add events to Apple Calendar and Google Calendar using voice commands or text.

**Examples:**
- 🎤 "Add gym session tomorrow at 6 PM" → Creates calendar event
- 💬 "Meeting with team next Monday 2pm" → Schedules automatically
- 📱 Works with both Apple Calendar and Google Calendar
